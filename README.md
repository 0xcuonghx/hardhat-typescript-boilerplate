# Hardhat Boilerplate

- [x] Hardhat
- [x] Typescript
- [x] Solhint
- [x] Eslint
- [x] Prettier

# Command

## Hardhat

```
npx hardhat test
npx hardhat clean
npx hardhat compile
npx hardhat run scripts/deploy.ts
```

## Solhint

```
npx solhint 'contracts/**/*.sol'
npx solhint 'contracts/**/*.sol' --fix
```

## Eslint

```
npx eslint '**/*.{js,ts}'
npx eslint '**/*.{js,ts}' --fix
```

## Prettier

```
npx prettier '**/*.{json,sol,md}' --check
npx prettier '**/*.{json,sol,md}' --write
```
