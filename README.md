# Hardhat Boilerplate

- [x] Hardhat
- [x] Typescript
- [x] Solhint
- [x] Eslint
- [x] Prettier
- [x] NatSpec

# Command

## Hardhat

```
npx hardhat test
npx hardhat clean
npx hardhat compile
npx hardhat run scripts/deploy.ts
```

## Solhint

```
npx solhint 'contracts/**/*.sol'
npx solhint 'contracts/**/*.sol' --fix
```

## Eslint

```
npx eslint '**/*.{js,ts}'
npx eslint '**/*.{js,ts}' --fix
```

## NatSpec
```
npx hardhat docgen
```
